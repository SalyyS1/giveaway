BOT DISCORD ĐẦY ĐỦ

Các tính năng bao gồm:
- Hệ thống ticket
- Theo dõi lời mời
- Quản lý (cấm, kick, cảnh cáo, tắt tiếng)
- Giveaways (tặng quà)
- Thông báo live Twitch, Kick, YouTube, Dailymotion, Snapchat

Hướng dẫn:
1. Cài đặt Node.js 18+
2. Đổi tên .env.example thành .env và thêm TOKEN của bạn
3. Chạy với: npm install && node index.js
